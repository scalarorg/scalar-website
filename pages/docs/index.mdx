# What is Scalar?

<p class="text-lg">Scalar - Unified Shared Liquidity & Security Layer.</p>

![](/docs/introduction/what_is_scalar.png)

Scalar is the first blockchain built on Scalaris - a novel framework designed to facilitate the rapid development of decentralized applications by leveraging advanced consensus protocols.

Scalar is expected to be the most performant EVM and Move compatible, high throughput, interoperable smart contract platform in the space ever seen to date.
